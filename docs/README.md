# 今日专注支持页面

这是今日专注 App 的 GitHub Pages 静态支持站点，包含：

- `index.html`：技术支持主页
- `privacy.html`：隐私政策
- `terms.html`：用户协议

GitHub Pages 发布建议：

1. 打开 GitHub 仓库的 `Settings`。
2. 左侧选择 `Pages`。
3. `Source` 选择 `Deploy from a branch`。
4. `Branch` 选择 `main`，目录选择 `/docs`。
5. 保存后等待 GitHub Pages 发布。

发布后可用于 App Store Connect：

- 技术支持网址：`https://<你的 GitHub 用户名>.github.io/<仓库名>/`
- 隐私政策 URL：`https://<你的 GitHub 用户名>.github.io/<仓库名>/privacy.html`
